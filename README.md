# Java Collections: HashSet & ArrayDeque Practice

This repository contains solutions for 21 programming tasks focusing on the Java Collections Framework.

## 📌 Project Overview

### Part 1: HashSet (Tasks 1-10)
I implemented solutions for managing unique data using HashSet. 
* *Key Operations:* Bulk additions (addAll), removals (removeAll), and finding common elements (retainAll).
* *Logic:* Leveraged the O(1) time complexity of HashSets for efficient membership checking and deduplication.

### Part 2: ArrayDeque (Tasks 11-21)
I used ArrayDeque to demonstrate both Stack (LIFO) and Queue (FIFO) behaviors.
* *Palindrome Check:* Implemented by comparing characters from both ends of a deque (removeFirst and removeLast).
* *Task Scheduler:* A priority system where high-priority tasks are added to the front (addFirst) and regular tasks to the back (addLast).

## 🛠 Tech Stack
* *Language:* Java 17+
* *Data Structures:* HashSet, ArrayDeque
